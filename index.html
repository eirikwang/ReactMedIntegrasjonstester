React

This article contains my experiences in setting up a react.js procject with a bit more complexity than Hello world

We will discuss the following subjects:

  • React - putting it together
  • React-router for application routing
  • Redux for application state
  • React-Intl for localisation
  • Mocha for unit testing
  • Webdriver/Nightwatch.js for integration tests
  • Express for mocking server calls for the integration tests
  • Gulp/npm with browserify for build (html, js, css and minify) (I miss maven sometimes)
